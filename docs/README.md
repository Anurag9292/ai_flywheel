# AI Flywheel - Documentation

**ai_flywheel** — *one flow to rule them all.*

AI Flywheel is a Python-based platform that serves as a personal startup operating system for a solo founder to rapidly spin up and scale multiple AI-native businesses. It provides ~30 shared utility modules (agents, tools, experiments, prompts, data pipelines, orchestration) as a foundation layer, with domain-specific orchestrators composed on top for each new venture — HR, sales, knowledge management, and beyond.
